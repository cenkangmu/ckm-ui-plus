### Props 属性
| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| :----: | :----: | :----: | :----: | :----: |
| value | 当前进度 | Number | - | 0 |
| max | 最大值 | Number | - | 100 |
| size | 宽度、高度 | String | - | 50px |
| color | 字体颜色 | String | - | #333 |
| fontSize | 字体大小 | String | - | 12px |
| lineColor | 线条颜色 | String | - | #eee |
| activeColor | 进度达到的线条颜色 | String | - | #999 |
| lineWidth | 线条宽度 | Number | - | 3 |
