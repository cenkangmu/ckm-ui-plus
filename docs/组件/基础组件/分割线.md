### Props 属性

| lineHeight | 高度 | String | - | 1px |
| lineColor | 线条颜色 | String | - | #e5e5e5 |
| textColor | 字体颜色 | String | - | #ccc |
| fontSize | 字体大小 | String | - | 12px |
