### Props 属性
| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| :----: | :----: | :----: | :----: | :----: |
| width | 宽度 | String | - | - |
| height | 高度 | String | - | 50px |
| background | 背景颜色 | String | - | #fff |
| color | 文本颜色 | String | - | #333 |
| fontSize | 字体大小 | String | - | 14px |
| borderColor | 底部边框颜色 | String | - | #f5f5f5 |

### Slots 插槽
| 插槽名称 | 说明 |
| :----: | :----: |
| right | 标签栏右侧，当存在此插槽时，将替换右侧箭头 |
